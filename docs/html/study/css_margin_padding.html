margin 속성

현재 요소 주변의 여백을 뜻함

마진을 이용하면 요소와 요소간의 간격 조절 가능

margin: 크기 | 백분율 | auto

종류 설명 예시
크기 너비값이나 높이값을 px, em 같은 단위로 지정 margin: 10px;
백분율 박스 모델을 포함한 부모 요소를 기준으로 너비값이나 높이값을 비율로 지정 margin: 01.%
auto display 속성에서 지정한 값에 맞게 적절한 값을 자동으로 지정 margin: auto

margin 속성으로 가운데 정렬하기

배치할 요소의 너비값이 정해져 있어야 함

margin-left와 margin-right의 속성값을 auto로 지정

style="width: 300px; margin: auto auto;"

마진 중첩 현상

요소를 세로로 배치할 경우 마진과 마진이 만날 때 마진 값이 큰 쪽으로 정해지는 것

2개의 요소가 있을 경우 각 요소간의 마진이 20px, 30px 일 경우,
합쳐서 50px 가 마진이 아닌, 두 요소간의 여백은 30px 가 됨
