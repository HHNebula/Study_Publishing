테두리 스타일 지정하기

박스 모델의 값 지정 방향

4개 방향을 한번에 지정할 때는 방향 순서를 지켜야 함

top > right > bottom > left (12시 부터 시계방향)

border-style 속성

테두리를 표시하기 위해 먼저 스타일 지정 필요

종류 설명
none 테두리를 표시하지 않음, 기본값
hidden 테두리를 감춤,
표에서 border-collapse: collapse 일 경우
다른 테두리도 표시되지 않음
solid 실선 테두리 표시
dotted 점선 테두리 표시
dashed 짧은 직선 테두리 표시
double 테두리를 이중 선으로 표시
두 선 사이의 간격이 border-width 값이 됨
groove 테두리를 창에 조각한 것 처럼 표시
홈이 파인듯 한 입체감
insert 표에서 border-collapse: collapse 일 경우 groove와 동일함
표에서 border-collapse: seperate 일 경우 전체 박스 테두리가 창에 박혀있는 것 같이 표현
outset 표에서 border-collapse: collapse 일 경우 ridge 동일함
표에서 border-collapse: seperate 일 경우 전체 박스 테두리가 창에서 튀어나온 것 같이 표현
ridge 테두리를 창에서 튀어나온 것 같이 표현

style="border-style: dotted;"

style="border-style: dashed;"

style="border-style: groove;"

style="border-style: outset;"

border-width 속성

테두리 두께 지정

크기나 thin, medium, thick 으로 설정

style="border-style: solid; border-width: 3px;

style="border-style: solid; border-width: 5px;

style="border-style: solid; border-width: thin;

style="border-style: solid; border-width: medium;

style="border-style: solid; border-width: thick;

border-color 속성

테두리 색상 지정

style="border-color: red; border-style: solid;"

style="border-top-color: red; border-bottom-color: blue; border-style: solid;"

style="border-color: red blue green orange; border-style: solid;"

border 속성

테두리 스타일, 두께, 색상을 한번에 표시

순서는 상관 없음

style="border: dashed 3px red;"aaaaaaaaa

border-radius 속성

박스 모델의 테두리를 둥글게 처리

크기 또는 백분율로 표시

이미지 파일에도 적용 가능


style="border-radius: 10px

style="border-radius: 10%

boder와 radius 사이에 위치를 나타내는 예약어를 사용하면 꼭짓점 마다 다르게 처리 가능함

예약어는 top-left, top-right, botton-left, bottom-right

style="border-top-left-radius: 20%; border-bottom-right-radius: 20%;"
