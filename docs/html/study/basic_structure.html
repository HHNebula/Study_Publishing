<!DOCTYPE html>

웹 브라우저에게 이제부터 처리할 문서는 HTML 문서라고 알려주는 것

<html>

웹 문서 시작을 알리는 태그

실제 문서 정보와 내용이 시작되고 끝나는 것을 표시

lang 속성을 사용해 문서에서 사용할 언어 지정

문서 정보를 지정하는 head 부분과 내용을 입력하는 body 부분을 포함

<head>

브라우저에게 정보를 주는 태그

문서 관련 정보 입력, 웹 브라우저 화면에는 보이지 않음

문서에서 사용할 외부 파일 링크

문자 세트 등 문서 정보가 들어있는 meta 태그

문서 제목을 설정하는 title 태그 등 여러 정보 지정

<body>

웹 브라우저에 내용을 표시하는 태그

실제 브라우저에 표시될 내용 입력

대부분의 태그가 body 태그에서 사용되는 태그들
