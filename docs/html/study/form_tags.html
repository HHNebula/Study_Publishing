폼이란?

사용자가 웹 사이트로 정보를 보낼 수 있는 요소

폼과 관련된 대부분의 작업은 정보를 입력받아 저장하거나 검색, 수정하는 작업

이러한 작업은 데이버 테이스를 기반으로 함

아이디를 입력하는 텍스트 필드나 버튼 등의 형태는 HTNL 에서 작성

입력된 정보를 처리하는 것은 ASP 나 PHP 같은 서버 프로그래밍 이용

form Tag 속성

속성 설명
method 사용자가 입력한 내용을 서버쪽으로 어떻게 넘겨줄 것인지 지정
get - 데이터를 256 ~ 4,096 byte 까지 전달 가능, 사용자 입력 값이 주소 표시줄에 드러남
post - 입력한 내용의 길이에 제한이 없음, 사용자 입력 값이 주소 표시줄에 보여지지 않음
name 자바스크립트로 폼을 제어할 때 사용할 폼의 이름을 지정
action form 태그 안의 내용을 처리해 줄 서버 프로그램을 지정
target action 속성에서 지정한 스크립트 파일을 현재 창이 아닌 다른 위치에서 열도록 함

fieldset Tag / legend Tag

fieldset - 폼 요소를 그룹으로 묶는 태그

legend - 그룹으로 묶는 구역에 제목을 붙이는 태그

<fieldset>

<legend>그룹 이름</legend>

</fieldset>

label Tag

input 태그와 같은 폼 요소에 레이블을 붙일 때 사용

label 태그의 for 속성과 input 태그의 id 속성을 동일하게 작성하여 연결함

input 태그의 주요 타입

종류 설명 추가 속성
text 한 줄짜리 텍스트 입력 박스
주로 아디나 이름, 주소 등 텍스트 입력		 size = 화면에 보여지는 필드의 길이
value = 텍스트 필드 부분에 보여질 내용, password 는 X
maxlength = 입력 가능한 최대 문자 수정하는
password 비빌번호 입력 필드, 입력값이 문서에 보여지지 않음
사용자가 입력하는 내용이 * 또는 • 로 보여짐
search 검색할 때 입력하는 필드
검색 창에 x 가 표시되어 검색어 삭제 쉬움
url url 주소를 입력할 수 있는 필드
email 이메일 주소를 입력할 수 있는 필드
메일 주소 형식 자동 체크
tel 전화번호를 입력할 수 있는 필드
사용자 입력을 체크하지는 않음
checkbox 주어진 여러 항목에서 다수를 선택할 수 있는 체크 박스 name = 버튼이 여러개 있을 경우 구분하기 위한 이름, 관련 있는 그룹 끼리 동일한 name 사용
value = 선택한 버튼을 서버에 알려줄 때 넘길 값, 영문 또는 숫자여야 함, 필수값
checked = 선택이 되어 있는 상태로 출력
radio 주어진 여러 항목에서 하나만 선택할 수 있는 라디오 버튼
number 숫자를 조절할 수 있는 스핀 박스 min = 필드에 입력할 수 있는 최소값, 기본값은 0
max = 필드에 입력할 수 있는 최댓값, 기본값은 100
step = 숫자 간격 조절, 기본값은 1
value = 필드에 표시할 초기값
range 숫자를 조절할 수 있는 슬라이드 막대
date 사용자 지역을 기준으로 연월일 날짜를 삽입 type="date | month | week" 3가지 중 하나 선택
연월일 모두, 연월 까지만, 주 단위로 선택
month 사용자 지역을 기준으로 연월 날짜를 삽입
datetime 국제 표준시로 설정된 날짜와 시간 삽입
-연월일시분초분할초		 type="time | datetime | datetime-local" 3가지 중 하나 선택
시간 만 입력할 지, 날짜도 입력할지 선택,
datetime-local은 사용자 지역에 맞는 날짜와 시간
datetime-local 사용자가 있는 지역을 기준으로 날짜와 시간 삽입
-연월일시분초분할초
submit 전송 버튼 value 속성을 이용해 버튼 표시 내용 지정
type="image" 사용 시 submit 버튼 대신 이미지 삽입
reset 리셋 버튼
button 일반 버튼 기능 없이 버튼 형태만 삽입
주로 자바스크립트 실행 시 사용
value 속성으로 버튼 표시 내용 지정
file 파일 첨부 버튼 파일 선택이나 찾아보기 버튼으로 표시됨
hidden 사용자에게 보이지는 않지만 서버로 넘겨주는 값이 있는 필드 value 속성으로 서버로 넘길 값 지정

input 태그의 속성

속성 내용
autofocus 페이지를 불러오자마자 원하는 폼 요소에 마우스 커서 표시
placeholder 입력란에 표시하는 힌트로 클릭 시 사라짐
readonly 내용을 보기만 하고 입력하지 못하게 함
required 필수 필드 체크, 브라우저에서 직접 체크함

form 에서 사용하는 여러 태그

태그 내용 / 속성
textarea 게시판 글 입력 양식, 사용자 약관 등
select / optgroup / option 여러 옵션 중에서 드롭다운 선택
select size - 화면에 표시할 드롭다운 개수
multiple - 드롭다운 목록에서 둘 이상의 항목을 선택할 때 사용
option value - 해당 항목을 선택할 때 서버로 넘겨줄 값
selected - 드롭다운 메뉴를 삽입할 때 기본적으로 선택해서 보여줄 항목
datalist, option 데이터 목록에 제시한 값중에서 선택 시 그 값이 자동으로 텍스트 필드에 입력됨
데이터 목록에 id를 이용해 이름을 붙이고, input 태그의 list 속성에 데이터 목록 id를 지정함
button 다양한 형태의 버튼 삽입
화면 낭독기에서 버튼임을 정확히 전달
CSS를 통해 원하는 형태로 꾸미기 가능
