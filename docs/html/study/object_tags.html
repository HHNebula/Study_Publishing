object Tag

기본형 : <object width="너비" height="높이" data="파일"></object>

음악 파일 뿐만 아니라 동영상이나 자바 애플릿, PDF 파일 등 다양한 개체를 삽입

data 속성 - 재생할 파일을 지정

width, height 속성 - 플레이어 화면 크기 지정

embed Tag

기본형 : <embed src="파일경로" width="너비" height="높이">

대부분 브라우저에서 사용 가능

object, audio, video 태그를 지원하는 브라우저에서 멀티미디어 삽입 시 사용

닫는 태그 없음

src 속성 - 멀티미디어 지정

width, height 속성 - 플레이어 크기 지정

audi, video Tag

기본형 : <audio src="파일경로"></audio>
기본형 : <video src="파일경로"></video>

audio, video 태그의 속성

속성 설명
controls 플레이어 화면에 컨트롤 바 표시
autoplay 오디오나 비디오를 자동으로 실행
loop 오디오나 비디오를 반복 재생
muted 오디오나 비디오의 소리를 제거
preload 페이지를 불러올 때 오디오나 비디오 파일을 어떻게 로딩할 것인지 지정
width, height 비디오 플레이어의 너비와 높이를 지정, 둘 중 하나만 지정하여도 자동 계산
poster="파일" video 태그에서 사용하는 속성, 재생되기 전 썸네일 이미지 지정
