color 속성

글자 색 지정, 16진수 값이나 HSL값, 색상 이름 중에서 사용

p style="color: red;"

p style="color: rgb(50, 46, 255);"

p style="color: rgba(164, 255, 142, 0.329);"

p style="color: #192842;"

p style="color: hsl(240, 100%, 50%);"

  • 색상 이름 표기법
    • 잘 알려진 색상 이름으로 표시 (red, blue 등)
    • 모든 브라우저에서 표현할 수 있는 색상
  • hsl/hsla 표기법
    • style="color: hsl(240, 100%, 50%);"
    • 앞의 숫자부터 색상, 채도, 밝기의 양
    • 투명도 조절시 알파값 추가, 0~1 값 중 사용
    • style="color: hsla(240, 100%, 50%, 0.5);"
  • 16진수 표기법
    • #ffffff 처럼 #과 함께 16진수로 표시
    • 앞에서부터 두자리 씩 묶어 빨강, 초록, 파랑의 양
    • 두자리씩 중복될 경우 생략 가능 #ffffff == #fff
  • rgb/rgba 표기법
    • style="color: rgb(50, 46, 255);"
    • 세자리의 숫자로 표시, 앞에서부터 빨강, 초록, 파랑의 양
    • 하나도 섞이지 않으면 0, 가득 섞였을 경우 255
    • 투명도 조절시 알파값 추가, 0~1 값 중 사용
    • style="color: rgba(65, 49, 44, 0.863);"

text-align 속성

텍스트 정렬 방법 지정

style="text-align: start;"
현재 텍스트 줄의 시작 위치에 맞추어 문단 정렬

style="text-align: end;"
현재 텍스트 줄의 끝 위치에 맞추어 문단을 정렬

style="text-align: left;"
왼쪽에 맞추어 문단을 정렬

style="text-align: right;"
오른쪽에 맞추어 문단을 정렬

style="text-align: center;"
가운데에 맞추어 문단을 정렬

style="text-align: justify;"
양쪽에 맞추어 문단을 정렬

line-height 속성

문단의 줄 간격 지정

숫자와 백분율은 부모 요소를 기준으로 몇 배인지 지정

보통 글자 크기의 1.5 ~ 2배 정도면 적당함

줄 간격은 텍스트를 세로 정렬할 때도 유용함
->line-height의 속성 값을 영역의 height의 값과 똑같이 지정

text-decoration 속성

텍스트의 밑줄을 긋거나 가로지르는 줄 표시

텍스트 링크의 밑줄을 없앨 때도 사용

p style="text-decoration: none;"
밑줄을 표시하지 않음

p style="text-decoration: underline;"
밑줄을 표시함

p style="text-decoration: overline;"
윗줄을 표시함

p style="text-decoration: line-through;"
영역을 가로지르는 취소선을 표시함

text-shadow 속성

텍스트에 그림자 효과를 추가하는 속성

text-shadow: none | 가로거리 세로거리 번짐 정도 색상

p style="text-shadow: 5px 5px 3px gray;"

p style="text-shadow: 5px 5px 3px gray;"

text-transform 속성

영문 텍스트의 대문자나 소문자를 바꾸는 속성

text-transform: none | capitalize | uppercase | lowercase

p style="text-transform: none;"
변환하지 않음

p style="text-transform: capitalize;"
시작하는 첫 번째 글자를 대문자로 변환

p style="text-transform: uppercase;"
모든 글자를 대문자로 변환

p style="text-transform: lowercase;"
모든 글자를 소문자로 변환

letter-spacing, word-spacing 속성

글자간 간격, 단어간 간격

letter-spacing: nomal | 크기

word-spacing: nomal | 크기

p style="letter-spacing: 2px;"
자간 크기 2px

p style="word-spacing: 5px;"
단어 간 크기 5px
