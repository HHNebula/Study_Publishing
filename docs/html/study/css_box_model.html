박스 모델

실제 콘텐츠 영역, 패딩(padding), 박스의 테두리(border), 그리고 마진(margin) 등의 요소로 구성됨

width, height 속성

박스 모델 중 실제 콘텐츠 영역(가운데) 크기 지정

width 200px
height 200px
고정 크기

width 20%
height 20%
가변 크기

width auto
height auto
콘텐츠 양에 따라 자동 결정, 기본값

box-sizing 속성

실제 박스 모델의 너비를 계산할 때 어디까지 포함할지 결정

style="box-sizing: border-box;"
테두리까지 포함해서 너비값을 지정

style="box-sizing: content-box;"
콘텐츠 영역만 너비값 지정, 기본값

box-shdow 속성

선택한 요소에 그림자 효과

box-shadow: 수평거리 수직거리 흐림정도 번짐정도 색상 inset
수평거리와 수직거리는 필수, 나머지는 옵션

  • 수평거리 : 그림자가 가로로 얼마나 떨어져 있는지, 양수는 오른쪽, 음수는 왼쪽
  • 수직거리 : 그림자가 세로로 얼마나 떨어져 있는지, 양수는 아래쪽, 음수는 위쪽
  • 흐림정도 : 생략시 0 기본값, 진한 그림자를 표시함, 커질수록 부드러운 그림자
  • 번짐정도 : 양수값 사용 시 모든 방향으로 그림자가 퍼짐, 음수값은 축소됨, 0이 기본값
  • 색상 : 한 가지 혹은 공백으로 구분하여 여러 색상 지정 가능, 기본값은 black
  • insert : 설정 시 안쪽 그림자로 그려짐

p style="box-shadow: 2px -2px 5px 0px;"

p style="box-shadow: 5px 5px 15px 5px blue;"
