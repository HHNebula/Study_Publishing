<ol>, <li> 태그 - 순서 있는 목록

ol 태그를 연 뒤 그 안에 li 태그를 작성

  1. 항목 1
  2. 항목 2
  3. 각 항목 앞에 숫자가 붙여짐
  4. type 속성으로 순서 목록의 숫자 조정
  5. 1 : 숫자 / a : 소문자 / A : 대문자 / i : 로마 소문자 / I : 로마 대문자
  6. start 속성으로 목록의 시작 번호 수정
  1. type="I" start="3"
  2. type="I" start="3"
  3. type="I" start="3"
  1. type="A" start="7"
  2. type="A" start="7"
  3. type="A" start="7"

<dl>, <dt>, <dd> 태그 - 설명 목록

dl 태그를 연 뒤 그 안에 dt와 dd 태그를 작성

dt는 설명의 제목, dd는 설명의 내용을 작성함

하나의 dt에 여러개의 dd값을 가질 수 있음
