display 속성

요소의 배치 방법을 결정하는 속성

종류 설명
block 인라인 레벨 요소를 블록 레벨 요소로 만듬
inline 블록 라인 요소를 인라인 레벨 요소로 만듬
inline-block 인라인 레벨 요소와 블록 레벨 요소의 속성을 둘다 가짐
마진과 패딩을 지정할 수 있음
none 해당 요소를 화면에 표시하지 않음

flaot 속성

요소를 왼쪽이나 오른쪽에 떠 있게 만드는 속성

float: left | right | none

종류 설명
left 해당 요소를 문서의 왼쪽으로 배치
right 해당 요소를 문서의 오른쪽으로 배치
none 좌우 어느쪽으로도 배치하지 않음

style="float: left; margin: 10px;"

왼쪽에 떠있게 만들어 우측에 다른 태그가 올 수 있음






clear 속성

float 속성을 무효화 시키는 속성

clear: none | left | right | both

선택한 위치의 float 를 해제함

left, right, bottom, top 속성

웹 요소의 위치를 지정하는 속성

종류 설명
left 기준 위치와 요소 사이에 왼쪽으로 얼마나 떨어져 있는지
right 기준 위치와 요소 사이에 오른쪽으로 얼마나 떨어져 있는지
top 기준 위치와 요소 사이에 위쪽으로 얼마나 떨어져 있는지
bottom 기준 위치와 요소 사이에 아래쪽으로 얼마나 떨어져 있는지

position 속성

웹 문서 안에 요소들을 배치하기 위한 속성

position: static | relative | absolute | fixed

종류 설명
static 요소를 문서의 흐름에 맞추어 배치
relative 이전 요소에 자연스럽게 연결해 배치하되 위치를 지정할 수 있음
absolute 원하는 위치를 지정해 배치
fixed 지정한 위치에 고정 배치, 요소가 잘릴 수 있음
