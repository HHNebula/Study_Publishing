연결 선택자란?

선택자와 선택자를 연결해 적용 대상을 제한하는 선택자

컴비네이션 선택자 혹은 조합 선택자 라고도 함
하위 선택자, 자식 선택자, 인접 형제 선택자, 형제 선택자 가 해당됨

하위 선택자

부모 요소에 포함된 모든 하위 요소에 스타일이 적용됨
자식 요소 뿐만 아니라 손자 요소, 손자의 손자 요소 등 모든 하위 요소까지 적용
하위 선택자를 정의할 때는 상위 요소와 하위 요소를 나란히 써야함

기본 구조 : 상위요소 하위 요소 { 스타일 }

div p { ... } = div 태그의 아래에 있는 모든 p 요소

.contents div { ... } = contents 클래스의 아래에 있는 모든 p 요소

자식 선택자

자식 요소에 스타일을 적용하는 선택자
해당하는 자식 요소에만 스타일 적용

기본 구조 : 부모 요소 > 자식 요소 { 스타일 }

div > p { ... } = div 요소의 자식인 p 요소에만 스타일 적용

.contents > div { ... } = contents 클래스 요소의 자식인 p 요소에만 스타일 적용

인접 형제 선택자

같은 부모를 가진 형제 요소 중 첫 번째 동생 요소에만 스타일 적용
요소1 과 요소2 사이에 + 기호 사용
요소1 과 요소2 는 같은 레벨이면서 요소1 이후 첫 번째 요소2 만 적용

기본 구조 : 요소1 + 요소2 { 스타일 }

h1 + p { ... } : h1 요소 다음에 있는 p 요소 중 첫 번째 요소에만 스타일 적용

형제 선택자

인접 형제 선택자와 달리 모든 형제 요소에 스타일 적용

기본 구조 : 요소1 ~ 요소2 { 스타일 }

h1 ~ p { ... } = h1 요소 다음에 오는 모든 형제 p 요소에 적용

속성 선택자

지정한 속성 혹은 속성과 값을 가진 요소를 찾아 스타일 적용

a[href] { ... } = a 태그 중에서 href 속성이 있는 요소만 적용

a[target="_blank"] { ... } = a 태그 중에서 target 속성의 값이 _blank인 요소만 적용

[class ~= "button"] { ... } = class 속성에 button 값이 포함되어 있는 요소를 찾아 적용

a[title |= us] { ... } = title 속성 값이 us 이거나 us로 시작하는 a 요소를 찾아 적용

a[title ^="eng"] { ... } = title 속성 값이 eng 로 시작하는 a 요소를 찾아서 적용

a[href $= hwp] { ... } = href 속성값이 hwp로 끝나는 a 요소를 찾아 스타일 적용

a[href *= w3] { ... } = href 속성값 중 일부에 w3 가 있는 a 요소를 찾아 스타일 적용
