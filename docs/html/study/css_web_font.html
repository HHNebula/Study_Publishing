웹 폰트 사용하기

웹 폰트란

  • 웹 문서 안에 글꼴 정보도 함께 저장
  • 사용자가 웹 문서 접속 시 글꼴을 사용자 시스템으로 다운로드시켜 사용하는 글꼴
  • @font-face 속성 사용

구글 폰트 사용하기

  1. https://fonts.google.com 로 접속
  2. 한글 폰트 검색
  3. 원하는 폰트 선택 후 하단 select 클릭
  4. 우측 상단 네모 버튼 클릭
  5. Link 항목에 있는 소스 복사
  6. 웹 문서의 head 태그 안에 붙여넣음
  7. CSS rules to specify families 항목 복사
  8. font-family 속성 붙여넣기, 웹 폰트 글꼴 이름 사용
