글꼴 관련 스타일

font-family 속성

font-family:글꼴이름 | [글꼴이름, 글꼴이름]

선택자 { font-family: '맑은 고딕', 돋움, 굴림 }

style="font-family:'Courier New', Courier, monospace;"

style="font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;"

  • 웹 문서에서 사용할 글꼴을 지정
  • body, p, hn 태그 등 텍스트를 사용하는 요소에서 사용
  • 지정한 글꼴이 없을 경우에 대비해 두 번째, 세 번째 글꼴까지 지정
  • 둘 이상의 글꼴 이름을 지정할 때는 쉼표로 구분
  • body 태그 스타일에서 한 번 정의하면 전체에 적용되고 문서 안의 모든 자식 요소에 계속 같은 글꼴이 사용됨

font-size 속성

fonr-size: 절대 크기 | 상대 크기 | 크기 | 백분율

  • 절대 크기 : 브라우저에서 지정한 글자 크기
  • 상대 크기 : 부모 요소의 글자 크기를 기준으로 상대적인 글자 크기를 지정
  • 크기 : 브라우저와 상관없이 글자 크기를 직접 지정
  • 백분율 : 부모 요소의 글자 크기를 기준으로 백분율( % ) 로 표시
  • 키워드 사용 가능
    • xx-small
    • x-small
    • small
    • medium
    • large
    • x-large
    • xx-large
  • 단위를 사용해 글자 크기 지정
    • em : 부모 요소에서 지정한 대문자 M의 너비를 기준(1em)으로 한 후 비율값을 지정
    • rem : 문서 시작 부분에서 지정한 크기를 기준(1rem)으로 한 후 비율값을 지정
    • ex : 해당 글꼴의 소준자 x의 높이를 기준(1ex)으로 한 후 비율값을 지정
    • px : 모니터의 1픽셀을 기준(1px)로 한 후 비율값을 지정
    • pt : 포인트 라고 하며, 일반 문서에서 많이 사용

font-style 속성

font-style: nomal | iralic | oblique

style="font-style: normal;"

style="font-style: italic;"

style="font-style: oblique;"

font-weight 속성

font-weight: normal | bold | bolder | lighter | 100 | 200 | ... | 800 | 900

style="font-weight: normal;" 기본 값, 보통 굵기

style="font-weight: bold;" 굵게

style="font-weight: bolder;" 원래보다 더 굵게

style="font-weight: lighter;" 원래보다 더 가늘게

style="font-weight: 800;" 100~900 사이 입력, 클수록 더 굵게
