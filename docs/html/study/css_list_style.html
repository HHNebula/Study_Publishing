list-style-type 속성

순서없는 목록의 불릿이나 순서 목록의 숫자를 바꾸는 속성

ul / ol / li 태그에 적용 가능

  • style="list-style-type: disc;"
    채운 원 모양
  • style="list-style-type: circle;"
    빈 원 모양
  • style="list-style-type: square;"
    채운 사각형 모양
  • style="list-style-type: decimal;"
    1부터 시작하는 10진수
  • style="list-style-type: decimal-leading-zero;"
    앞에 0이 붙는 1부터 시작하는 10진수
  • style="list-style-type: lower-roman;"
    로마 숫자 소문자
  • style="list-style-type: upper-roman;"
    로마 숫자 대문자
  • style="list-style-type: lower-alpha;"
    알파벳 소문자
  • style="list-style-type: upper-alpha;"
    알파벳 대문자
  • style="list-style-type: none;"
    불릿이나 숫자를 없앰

list-style-image 속성

순서 없는 목록의 불릿을 이미지로 바꾸는 속성

  • style="list-style-image: url(../../imgs/ico.jpg);"
    • style="list-style-image: url(../../imgs/ico.jpg);"

list-style-position 속성

불릿 번호를 들여쓰거나 내어쓸 수 있음

  • li style="list-style-position: inside;"
  • li style="list-style-position: outside;"

list-style 속성

list-style-type, list-style-position, list-style-image 속성을 한번에 표시

  • li style="list-style: none;"
  • li style="list-style: lower-alpha inside;"
